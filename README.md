# ParkSmart — Vercel Ready Website

This folder is a complete deployable Next.js website/web app for ParkSmart.

## Important
Upload the contents of this folder directly to GitHub.
Do not upload the parent folder as a nested project.

Correct root files:
- package.json
- next.config.mjs
- vercel.json
- app/
- public/

## Local test

```bash
npm install
npm run dev
```

Open:
```txt
http://localhost:3000
```

## Vercel settings

Framework Preset:
```txt
Next.js
```

Root Directory:
```txt
./
```

Build Command:
```txt
npm run build
```

Install Command:
```txt
npm install
```

Output Directory:
Leave empty / default.

## Deploy

Import the GitHub repository into Vercel and click Deploy.
